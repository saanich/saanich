The Greater Victoria Public Library has four branches in Saanich, all branches have internet access and email available with a valid library card. By using the GVPL Web Site you can access other library services, such as the online catalogue or putting a book on hold.

Visit gvpl.ca for more information about GVPL's Branch locations and hours of operation.
