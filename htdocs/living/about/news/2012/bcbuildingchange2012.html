On December 20, 2012 the new 2012 BC Building Code comes into effect.

Province of BC’s website

Building and Safety Standards

All building and plumbing permit applications received by December 19, 2012 will be processed under the current 2006 BC Building Code.

All building and plumbing permits applied for on or after December 20th will be processed under the new 2012 BC Building Code.

For specific details about the changes to the BC Building Code, please visit the Province of BC’s website
